# Atmos | A weather app created with react
`Atmos` is a weather app that displays current weather along with hourly and daily forcast for the week in any city. This is made possible by fetching data from OpenWeather API and displaying it. It shows the current temperature in both metric and imperial, humidity, windspeed, how the temperature feels, highest temperature, lowest temperature and, sunrise and sunset time for a city.
## Technologies used
- [react](https://create-react-app.dev/docs/getting-started)
- [tailwindcss](https://tailwindcss.com/)
- [OpenWeather API](https://openweathermap.org/api)
